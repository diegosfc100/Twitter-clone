## :information_source: Como usar:

```shell
# Clone esse repositório:
$ git clone git@github.com:diegosfc100/Twitter-login-clone.git
```
```shell
# Vá para o repositório
$ cd Twitter-login-clone
```
```shell
# Inicie o servidor na linha de comando:
$ open index.html
```
